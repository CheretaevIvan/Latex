# Установка PSCYR для Latex

1. Копируем содержимое папки fonts/ в папку, установлен MiKTeX.

2. Нажимаем Пуск и Выполнить.

3.  Вводим там следующее:

	>initexmf --edit-config-file dvips
	
	Oткроется пустой файл config.ps
	
	Вводим следующий текст и сохраняем:

    >p +pscyr.map
	
4. Аналогично вводим команду:

	>initexmf --edit-config-file pdftex
	
    и в блокноте вводим и сохраняем текст:
	
    >+pscyr.map
	
5. Аналогично вводим команду:

    >initexmf --edit-config-file updmap
	
    и в блокноте вводим и сохраняем текст:

    >Map pscyr.map
	
6. Вводим команду:

    >updmap

    Появится черное окно и ждем когда оно само закроете. У меня заняло пару минут.

7. Вводим команду:

    > initexmf --edit-config-file dvipdfm

    и в блокноте вводим и сохраняем текст:

    > f pscyr2.map
	
8. Вводим команду:

    >initexmf -u

    Появится черное окно, и ждем, когда оно закроется. У меня прошло где-то полминуты.
	
9. Идем Пуск > Все программы > MiKTeX 2.9 > Maintenance(Admin) > Setting(Admin). Там в разделе General жмем Refresh FNDB.

10. Выполняем команду:

    >mkfntmap

# Итог

Теперь при добавлении пакета в tex файл:

```
\usepackage{pscyr} % Нормальные шрифты
```

Можно будет использовать те самые нормальные русские шрифты.
